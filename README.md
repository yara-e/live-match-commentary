# SportsLive ⚡ | Real-Time Match Dashboard

A full-stack, real-time sports dashboard featuring live score updates, match commentary, and paginated fixture lists.

## 🚀 Features

- **Live Real-Time Updates**: Uses WebSockets to push score changes and new commentary instantly.
- **Responsive Design**: Fully adaptive UI that works on Mobile, Tablet, and Desktop.
- **Smart Caching**: Frontend state management via TanStack Query for instant page transitions.
- **Seeder Script**: Includes a custom seeder to simulate live match events.

---

## 🛠 Tech Stack

### Frontend
- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Data Fetching**: TanStack Query (React Query)
- **Real-Time**: Native WebSockets

### Backend
- **Runtime**: Node.js + Express
- **Database**: PostgreSQL
- **ORM**: Drizzle ORM
- **Validation**: Zod
- **Real-Time**: `ws` (WebSocket library)
